# Images Directory

This directory contains images used for SEO and social media sharing.

## Required Files

1. **og-image.png** (1200x630px) - Open Graph image for social media sharing
   - Used by Facebook, Twitter, LinkedIn, etc.
   - Should include your logo and tagline
   - Recommended: "ProTenders - South Africa's Premier Government Tender Portal"

## Optional Files

1. **logo.png** (200x50px) - Main logo for structured data
2. **hero-image.jpg** - Hero image for homepage (optional)

## Image Guidelines

- **OG Image**: 1200x630px, PNG or JPG, under 300KB
- **Logo**: Transparent background PNG preferred
- All images should be optimized for web (compressed, WebP when possible)

## Next Steps

1. Create the OG image with your branding
2. Place it in this directory as `og-image.png`
3. The image will automatically be used in Open Graph and Twitter Card metadata

**Note**: The OG image is referenced in the root layout metadata but needs to be created from your design assets.
